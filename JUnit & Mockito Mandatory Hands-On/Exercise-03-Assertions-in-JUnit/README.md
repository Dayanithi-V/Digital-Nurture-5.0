# Exercise 3 - Assertions in JUnit

## Objective
To learn and implement different JUnit 5 assertion methods for validating Java programs.

## Problem Statement
Create a Calculator class and write JUnit test cases to verify its methods using different assertion methods.

## Assertions Covered
- assertEquals()
- assertNotEquals()
- assertTrue()
- assertFalse()
- assertNull()
- assertNotNull()
- assertThrows()

## Files
- Calculator.java
- CalculatorTest.java

## Learning Outcomes
- Understand different JUnit assertions.
- Validate expected and actual results.
- Test boolean conditions.
- Handle null checks.
- Test exception handling using assertThrows().
