# Exercise 7 - Implementing Constructor and Setter Injection

## Objective
Implement both Constructor Injection and Setter Injection using Spring IoC.

## Concepts Covered
- Spring IoC Container
- Constructor Injection
- Setter Injection
- XML Bean Configuration
- Dependency Injection (DI)

## Files
- pom.xml
- BookRepository.java
- BookService.java
- Main.java
- applicationContext.xml

## Output
```
Book Service
Book Repository
```

## Key Learning
- Constructor Injection injects dependencies through the constructor.
- Setter Injection injects dependencies using setter methods.
- Both approaches are supported by the Spring IoC Container.
