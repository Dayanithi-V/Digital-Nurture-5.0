# Exercise 3 - Stored Procedures

This exercise contains three stored procedures.

## Scenario 1
ProcessMonthlyInterest
- Adds 1% monthly interest to all savings accounts.

## Scenario 2
UpdateEmployeeBonus
- Updates employee salary based on the department and bonus percentage.

## Scenario 3
TransferFunds
- Transfers money from one account to another after checking the available balance.
